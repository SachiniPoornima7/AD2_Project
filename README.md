# Smart Parking Management System (SPMS)

This is a cloud-native, microservice-based application to manage real-time parking, payment, and user services.

## 📁 Microservices
- `user-service`
- `vehicle-service`
- `config-server`
- `eureka-server`
- `api-gateway`
